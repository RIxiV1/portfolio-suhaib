# 🚀 Shaik Mohammed Suhaib Portfolio

A premium, interactive developer portfolio showcasing AI-powered systems and modern web applications.

![Portfolio Preview](public/portfolio-preview.png)

## ✨ Overview

This portfolio is designed with a high-end aesthetic, featuring:
- **Glassmorphism UI**: A sleek, modern look using translucent layers and vibrant gradients.
- **Dynamic Animations**: Smooth, motion-driven interactions powered by Framer Motion.
- **AI-Focused Content**: Highlighting expertise in Agentic AI, LLM integration, and product thinking.
- **Interactive Elements**: Custom cursor, glowing section headers, and generative background paths.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (Canary)
- **Library**: React 19
- **Styling**: Tailwind CSS 4.2
- **Animation**: Framer Motion & Motion
- **Components**: Radix UI

## 🚀 Getting Started

1. **Install Dependencies**: `npm install`
2. **Run Dev Server**: `npm run dev`
3. **View Site**: [http://localhost:3000](http://localhost:3000)
